## 1. Vision / product goals

**"Battle Arena" / devf** — Flutter + Go social app where users challenge each other. Original elevator pitch: a **human competition engine layered on short-form media**. Three challenge archetypes:
- **Observation/Skill** ("find 10 ducks in a short") — timed, lowest time wins
- **Creative** ("who makes better sketch") — community-voted
- **Achievement** ("beat my score in Subway Surfers") — proof-based, fraud risk

Deeper product philosophy (from user's pasted ChatGPT convo they explicitly told me to internalize):
- "You're not ranking content — you're regulating human nervous systems."
- App promises **mood transition**, not content. Goal: user feels **better after 20 minutes, not worse**. That is the ethical-vs-dark-pattern test.
- 6 emotion buckets: **Joy / Relatable / Curiosity / Motivation / Emotional / Status-Ego**.
- Emotional wave pattern: `Joy → Relatable → Curiosity → Motivation → Neutral → Emotional → Joy` — adaptive per user.
- Time-of-day mood baseline: Morning=hopeful, Afternoon=distracted, Evening=social, Night=vulnerable.
- Scroll-gesture rule: "one gesture = one outcome forever".
- Zeigarnik effect (unfinished loops) used ethically — split stories, partial progress bars, "result revealed next".
- **Likes lie, watch time tells truth.**
- Scale target: **millions of users.** Sample data is only local scaffolding. Design every feature for scale from day one.

## 3. Backend recommendation algorithm (final architecture)

### Pipeline — `SmartFeedHandler`

```
1. Classify cohort (cold_start / new / engaged / power / at_risk)
2. Build session state (dopamine budget, skip streak, tried strategies, last categories/creators, resistance level)
3. multiSourceFetch — 5 parallel retrievers
     recency 0.35, trending 0.20, follow-graph 0.15, collab 0.15, embedding 0.15
4. scoreForUser per candidate — score = cohort-weighted blend of:
     social, freshness, energy fit, relevance, quality, novelty, tie strength,
     creator affinity, search boost, bounce penalty, negative creator penalty,
     sequence penalty (last categories + last creators), creator fatigue,
     recency decay exp(-ageDays/7), session continuity factor,
     LTR delta (tanh-clamped ±0.25) + Platt calibrated bonus (±0.15),
     two-tower cosine bonus (±0.20), bandit strategy bonus (soft-mixed)
5. Sort desc
6. filterUnseenScored — Redis ZSET per user (12h TTL, 2000 cap)
7. applyMMRDefault — λ=0.72, topK=40 (breaks near-duplicates)
8. detectLoop — 4 signals → untried-strategy override:
     SkipStreak≥4, category monoculture 3/4, creator flood 3/5, dopamine<0.15
9. buildSlotPattern — 12+ patterns incl. mood-driven
10. Respond; fire-and-forget markShownBatch + ltrStashBreakdownWithPos
```
